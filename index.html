In the directory with this index.html file are two subdirectories: server and client.

Each has a pre-made package.json with everything set up. Open a shell in each subdirectory and do:

npm install

Alternatively, you could move an old node_modules into each. Here is what must be present in each:

  • server: express sqlite sqlite3 cors
  • client: the node_modules from A11 (create-react-app + axios)

The server.js file already has all the needed module imports and configuration, you only have to add routes.

Start up the server with:

node (or nodemon) server.js

And the react client with:

npm run start

That should automatically open a new tab with the app and instructions. If it doesn't: http://localhost:3000
